# 破妄溯音

基于 Lombard 效应的实时语音通话活体检测系统前端。

## Features

- Product homepage and web console.
- Login and registration entry.
- Audio upload validation.
- Lombard response workflow.
- Liveness score, acoustic charts, feature deltas, reports, records.

## Development

```bash
cd src/frontend
pnpm install
pnpm dev
pnpm build
```
