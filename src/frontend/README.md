# 破妄溯音前端入口

## 页面目标

制作面向评委的作品主页和实时语音通话活体检测产品界面，突出 Lombard 效应、实时通话安全、活体置信度和抗语音伪造能力。

## 内容来源

- `submission/【项目报告】破妄溯音.pdf`
- `docs/project-brief.md`
- `docs/competition-form-outline.md`
- `docs/novelty-search-outline.md`
- `docs/submission-checklist.md`

## 页面模块

1. 首页首屏：作品名、AI 拟声诈骗痛点、实时通话安全价值、核心指标。
2. 通话控制台：音频流状态、噪声注入、语音采集、模型判断、预警结果。
3. 波形与频谱：普通语音和 Lombard 语音对比。
4. 活体结果：真人置信度、风险标签、历史检测记录。
5. 指标矩阵：准确率、响应延时、内存占用、VoIP 和网络环境表现。
6. 创新说明：实时通话场景、听觉反馈机制、双输入差分网络、多因素语料库。

## 视觉关键词

语音安全、频谱、波形、实时检测、风险预警、可信通话。
